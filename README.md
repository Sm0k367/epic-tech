## Quick Start

```bash
git clone https://github.com/Sm0k367/epic-tech.git
cd epic-tech
npm install
npm run dev
```

Then open `http://localhost:5173`

Add your keys in the UI or set them as environment variables:

```env
TAVILY_API_KEY=tvly-...
FIRECRAWL_API_KEY=fc-...
```
